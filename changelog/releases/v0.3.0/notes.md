The context bar widget now supports eight visual styles—blocks, lines, circles, parallelograms, diamonds, bars, stars, and specks—letting you match its look to your terminal aesthetic. The configuration screen is reorganized into Banner, General, and Widgets sections with tab navigation for faster access.

## 🚀 Features

### Configurable context bar styles

The context bar widget now offers eight visual styles. A new _context bar style_ setting in the configuration screen lets you pick the one you prefer.

| Style              | Used | Free | Reserved |
| ------------------ | ---- | ---- | -------- |
| `blocks` (default) | `■`  | `□`  | `▨`      |
| `lines`            | `━`  | `─`  | `┄`      |
| `circles`          | `●`  | `○`  | `◎`      |
| `parallelograms`   | `▰`  | `▱`  | `▰`      |
| `diamonds`         | `◆`  | `◇`  | `❖`      |
| `bars`             | `█`  | `░`  | `▒`      |
| `stars`            | `★`  | `☆`  | `✭`      |
| `specks`           | `•`  | `◦`  | `•`      |

Used cells are colored with the widget's icon color so they stand out from free and reserved cells, which stay dim.

*By @mavam and @claude.*

## 🔧 Changes

### Sectioned configuration screen with tab navigation

The configuration screen now organizes settings into three sections—Banner, General, and Widgets—instead of a single flat list. Use `Tab` and `Shift+Tab` to move between sections, while `↑`/`↓` still navigate items within and across sections. Setting descriptions are clearer and written from the user's perspective.

*By @mavam and @codex.*
