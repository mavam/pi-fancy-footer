The context bar widget now offers eight visual styles. A new _context bar
style_ setting in the configuration screen lets you pick the one you prefer.

| Style              | Used | Free | Reserved |
| ------------------ | ---- | ---- | -------- |
| `blocks` (default) | `■`  | `□`  | `▨`      |
| `lines`            | `━`  | `─`  | `┄`      |
| `circles`          | `●`  | `○`  | `◎`      |
| `parallelograms`   | `▰`  | `▱`  | `▰`      |
| `diamonds`         | `◆`  | `◇`  | `❖`      |
| `bars`             | `█`  | `░`  | `▒`      |
| `stars`            | `★`  | `☆`  | `✭`      |
| `specks`           | `•`  | `◦`  | `•`      |

Used cells are colored with the widget's icon color so they stand out
from free and reserved cells, which stay dim.
